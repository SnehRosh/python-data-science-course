# Data structures in python
## List✨✨
    - mutable
    - indexed
    - ordered
    - can contain duplicates
## Tuple
    - immutable
    - indexed
    - ordered
    - can contain duplicates
## Set
    - mutable
    - unindexed
    - unordered
    - can contain only unique values
## Dictionary✨✨
    - mutable
    - unindexed
    - ordered
    - can contain only unique keys
    - can contain duplicate values
